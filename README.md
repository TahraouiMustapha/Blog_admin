# 🛠️ Blog Admin App

An admin dashboard built as part of The Odin Project curriculum.  
This application allows administrator to manage blog content, including posts and comments, through a clean and functional interface.

---

## 🚀 Features

- View a list of all posts (published & unpublished)
- Publish / unpublish posts
- Create new posts
- Rich text editor integration (TinyMCE)
- Manage comments 
- Admin authentication
- Protected routes
- Image upload for posts

---

## 🧱 Tech Stack

- React
- React Router
- Tailwind CSS
- Fetch API
- TinyMCE (Rich Text Editor)
- JWT Authentication
